# first_lab
Key-Value Store з Redis
# Система кешування на базі Redis

Міні-проєкт, який демонструє роботу з Redis як з базою даних для кешування.

## Функціонал:
- **Керування користувачами:** перевірка авторизації перед доступом до кешу.
- **Підтримка різних типів даних:** кешування строк/чисел (String), об'єктів (Hash) та списків (List).
- **Збір метрик:** безпечний підрахунок та сортування популярних запитів за допомогою Redis Sorted Set.

## Як запустити:
1. Переконайтеся, що Redis сервер запущено на `localhost:6379`.
2. Встановіть залежності: `pip install redis`
3. Запустіть скрипт: `python main.py`
